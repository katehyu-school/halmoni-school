## 🎯 설계 철학

> **"한국 사람도 헷갈리는 포인트는 진지하게 다룰 것"**

큰 숫자(만, 억, 조)는 한국어 화자에게도 도전적임. 한국어 4자리 grouping vs 영어 3자리 grouping의 차이가 핵심 학습 장벽. 이걸 reference로 빼서 **언제든지 찾아볼 수 있게** 만드는 것이 목표.

## 📊 핵심 콘텐츠: 큰 숫자 (Large Numbers)

### 콘텐츠 1: 4자리 grouping 시각화 표

```
표기 (Notation)        한국어 grouping        읽기 (Reading)
─────────────────      ──────────────          ─────────────
1                  =>  1                       일
10                 =>  10                      십
100                =>  100                     백
1,000              =>  1000                    천
10,000             =>  1,0000  ⭐ 만 (key)     만
100,000            =>  10,0000                 십만
1,000,000          =>  100,0000                백만
10,000,000         =>  1000,0000               천만
100,000,000        =>  1,0000,0000 ⭐ 억(key)  억
1,000,000,000      =>  10,0000,0000            십억
10,000,000,000     =>  100,0000,0000           백억
100,000,000,000    =>  1000,0000,0000          천억
1,000,000,000,000  =>  1,0000,0000,0000 ⭐ 조  조
```

### 콘텐츠 2: 영어 vs 한국어 비교

#### 예시: 165,000원

**English approach (3-digit):**
```
165,000
│   └── thousand
└────── hundred
→ "One hundred sixty-five thousand"
```

**Korean approach (4-digit):**
```
16,5000
│   └── 천 (오천)
└────── 만 (십육만)
→ "십육만 오천"
```

### 콘텐츠 3: 인터랙티브 변환기 (선택)

학습자가 숫자 입력하면:
- 영어 grouping (3자리)
- 한국 grouping (4자리)
- 한국어 발음 자동 생성

예시:
```
입력: 1234567
영어: 1,234,567 (one million two hundred thirty-four thousand five hundred sixty-seven)
한국: 123,4567 (백이십삼만 사천오백육십칠)
```

---

## 💡 시각화 아이디어

### 1. 컬러 코딩 (4자리 단위)
- **만 단위** (4-7번째 자리): 파란색
- **억 단위** (9-12번째 자리): 보라색
- **조 단위** (13-16번째 자리): 빨간색

각 단위가 다른 색으로 보이면 grouping이 직관적.

### 2. "Key Unit" 강조 박스
- 만, 억, 조 = 한국어의 **자릿수 점프** 포인트
- 다른 자리는 십/백/천 반복
- 만/억/조에서 "새 단어" 사용

```
┌────────────────────────────────┐
│  만 (10,000) = KEY UNIT 1️⃣      │
│  십만(100,000) → 십 + 만        │
│  백만(1,000,000) → 백 + 만      │
│  천만(10,000,000) → 천 + 만     │
└────────────────────────────────┘
            ↓
┌────────────────────────────────┐
│  억 (100,000,000) = KEY UNIT 2️⃣ │
│  십억 → 십 + 억                 │
│  백억 → 백 + 억                 │
│  천억 → 천 + 억                 │
└────────────────────────────────┘
```

### 3. 단계별 분해 애니메이션 (선택)
- 큰 숫자 입력 → 4자리씩 분해 → 각 부분 한국어 변환 → 합치기
