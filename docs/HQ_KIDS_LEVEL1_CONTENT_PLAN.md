## 1과 — 안녕하세요? 👋

**학습 목표**: 기본 인사를 할 수 있고, 이름을 묻고 대답할 수 있다

### 핵심 문법
| 패턴 | 설명 | 예시 |
|------|------|------|
| **N이에요 / N예요** | "I am N" / "It is N" | 애라예요. / 리암이에요. |

> 💡 규칙: 받침 있으면 **이에요**, 받침 없으면 **예요**

### 핵심 표현
| 한국어 | English | 상황 |
|--------|---------|------|
| 안녕하세요? | Hello! (formal) | 어른에게 |
| 안녕? | Hi! (casual) | 친구에게 |
| 안녕히 가세요. | Goodbye (to one leaving) | 헤어질 때 |
| 안녕히 계세요. | Goodbye (to one staying) | 자리 뜰 때 |
| 고맙습니다. | Thank you | 감사할 때 |
| 이름이 뭐예요? | What is your name? | 이름 물을 때 |
| 저는 ___예요/이에요. | My name is ___. | 이름 대답 |

### 대화 예시 (애라 버전)
```
리암: 안녕? 이름이 뭐예요?
애라: 저는 애라예요. 이름이 뭐예요?
리암: 저는 리암이에요.
리암: 안녕히 가세요!
애라: 안녕히 계세요!
```

### 학습 어휘 — 이름 연습용
애라, 리암, 루카스, 마야, 카요, 올리비아

### 게임 아이디어
- **이름 카드 게임**: 친구 얼굴 카드를 보고 이름 맞히기 (N이에요/예요 정확히 사용)
- **몸동작 인사**: 절하기(안녕하세요), 손 흔들기(안녕!), 손 모으기(고맙습니다) 연결

### 문화 팁
> 🇰🇷 한국에서는 어른에게 **고개 숙여** 인사해요. 더 공손할 때는 **큰절**을 해요!

## 2과 — 이게 뭐예요? 🖊️

**학습 목표**: 교실 물건의 이름을 말할 수 있다

### 핵심 문법

| **이게 뭐예요?** | What is this? | 이게 뭐예요? |
| **저게 뭐예요?** | What is that (far)? | 저게 뭐예요? |
| **N이에요 / N예요** | It is N | 연필이에요. / 시계예요. |

### 핵심 어휘 — 교실 물건 (9개)
| 한국어 | English | 이미지 힌트 |

| 선생님 | teacher | 👩‍🏫 |
| 책상 | desk | 🪑 |
| 의자 | chair | 💺 |
| 칠판 | blackboard | 🖊️ |
| 시계 | clock | 🕐 |
| 지우개 | eraser | 🧹 |
| 책 | book | 📚 |
| 공책 | notebook | 📓 |
| 연필 | pencil | ✏️ |

### 확장 어휘 (보너스) — 문구용품
| 한국어 | English |
|--------|---------|
| 가위 | scissors ✂️ |
| 풀 | glue 🗂️ |
| 색종이 | colored paper 🌈 |
| 필통 | pencil case |
| 색연필 | colored pencils 🖍️ |
| 크레파스 | crayons 🖍️ |

### 대화 예시
```

선생님: 시계예요.
```

### 게임 아이디어

- **색칠 낱말 퍼즐**: 교실 물건 글자 찾아 색칠하기

## 3과 — 네, 빵이에요 🍞

**학습 목표**: 물건을 확인하고 네/아니요로 대답할 수 있다

### 핵심 문법
| 패턴 | 설명 | 예시 |
|------|------|------|
| **이게 N이에요?** | Is this N? | 이게 빵이에요? |
| **네, N이에요/예요.** | Yes, it is N. | 네, 빵이에요. |
| **아니요, N이에요/예요.** | No, it is N. | 아니요, 우유예요. |

### 핵심 어휘

**🍱 음식/간식 (6개)**
| 한국어 | English |
|--------|---------|
| 우유 | milk 🥛 |
| 주스 | juice 🧃 |
| 과자 | snack/crackers 🍘 |
| 빵 | bread 🍞 |
| 사탕 | candy 🍬 |
| 아이스크림 | ice cream 🍦 |

**🐾 동물 (4개)**
| 한국어 | English |
|--------|---------|
| 고양이 | cat 🐱 |
| 강아지 | dog 🐶 |
| 토끼 | rabbit 🐰 |
| 곰 | bear 🐻 |

### 대화 예시
```
리암: 이게 빵이에요?
애라: 네, 빵이에요.

마야: 이게 주스예요?
애라: 아니요, 우유예요.

카요: 이게 고양이예요?
루카스: 아니요, 강아지예요.
```

### 게임 아이디어
- **예/아니요 카드 게임**: 물건 카드 보여주며 "이게 ___이에요?" 질문 → 빠르게 네/아니요
- **스무고개**: 동물 카드 숨기고 네/아니요로만 맞히기

## 4과 — 우리 엄마예요 👨‍👩‍👧‍👦

**학습 목표**: 가족을 소개할 수 있다

### 핵심 문법
| 패턴 | 설명 | 예시 |
|------|------|------|
| **누구예요?** | Who is it? | 누구예요? |
| **우리 N이에요/예요.** | It's my N. | 우리 엄마예요. |
| **N이에요/예요.** | It's N. | 동생이에요. |

### 핵심 어휘 — 가족 (8개)
| 한국어 | English |
|--------|---------|
| 할아버지 | grandfather 👴 |
| 할머니 | grandmother 👵 |
| 아빠 | dad 👨 |
| 엄마 | mom 👩 |
| 나 | me/I 🙋 |
| 동생 | younger sibling 👶 |
| 우리 | our/my (family) |
| 친구 | friend 👫 |

### 대화 예시
```
올리비아: 누구예요?
애라: 우리 엄마예요.

루카스: 누구예요?
리암: 우리 할아버지예요.

카요: 누구예요?
마야: 동생이에요.
```

### 게임 아이디어
- **주사위 가족 게임**: 주사위 굴려서 나온 가족 칸에 가기 (뒤로/앞으로)
- **가족 앨범**: 가족 사진 가져와서 "누구예요?" 소개하기

### 문화 팁
> 🇰🇷 한국에서는 내 가족을 말할 때 **"우리 엄마", "우리 할머니"** 처럼 "우리"를 붙여요. "내 엄마" 대신 "우리 엄마" — 가족을 함께 묶는 표현이에요!

## 6과 — 학교에 가요 🚶

**학습 목표**: 어디에 가는지 말할 수 있다

### 핵심 문법
| 패턴 | 설명 | 예시 |
|------|------|------|
| **N은/는** | topic marker | 애라는 / 리암은 |

| **N에 와요.** | (I/someone) come to N. | 집에 와요. |
| **어디에 가요?** | Where are you going? | 어디에 가요? |

### 핵심 어휘 — 장소 (7개)
| 한국어 | English |
|--------|---------|
| 회사 | office/company 🏢 |
| 백화점 | department store 🛍️ |
| 유치원 | kindergarten 🧒 |
| 병원 | hospital/clinic 🏥 |
| 바다 | sea/ocean 🌊 |
| 산 | mountain ⛰️ |

```
마야: 어디에 가요?
```

## 8과 — 그림을 그려요 ✏️

**학습 목표**: 행동을 표현할 수 있다

### 핵심 문법
| 패턴 | 설명 | 예시 |
|------|------|------|
| **뭘 해요?** | What are you doing? | 뭘 해요? |
| **N을/를** | object marker | 그림을 / 우유를 |
| **N을/를 V해요.** | do V with N | 그림을 그려요. |

> 💡 받침 있으면 **을**, 없으면 **를**

### 핵심 어휘 — 행동 동사 (8개)
| 한국어 | English | 목적어 예시 |
|--------|---------|-----------|
| 봐요 | watch/look | 텔레비전을 봐요 📺 |
| 읽어요 | read | 책을 읽어요 📚 |
| 마셔요 | drink | 우유를 마셔요 🥛 |
| 먹어요 | eat | 빵을 먹어요 🍞 |
| 입어요 | wear/put on | 옷을 입어요 👗 |
| 그려요 | draw | 그림을 그려요 🎨 |
| 신어요 | wear (shoes) | 신발을 신어요 👟 |
| 써요 | write | 글씨를 써요 ✏️ |

### 대화 예시
```
엄마는 책을 읽어요.
아빠는 텔레비전을 봐요.
애라는 그림을 그려요.
리암은 우유를 마셔요.

마야: 뭘 해요?
카요: 옷을 입어요.
```

### 게임 아이디어
- **몸동작 게임**: 행동 카드 뽑아 몸으로 표현하면 → 나머지가 "책을 읽어요!" 맞히기
- **그림 완성**: 빈 그림 보고 "뭘 해요?" 질문에 답 쓰기

## 9과 — 사과를 좋아해요 🍎

**학습 목표**: 좋아하는 것과 싫어하는 것을 말할 수 있다

### 핵심 문법
| 패턴 | 설명 | 예시 |
|------|------|------|
| **N을/를 좋아해요.** | I like N. | 사과를 좋아해요. |
| **N을/를 싫어해요.** | I don't like N. | 떡을 싫어해요. |
| **뭘 좋아해요?** | What do you like? | 뭘 좋아해요? |
| **뭘 싫어해요?** | What don't you like? | 뭘 싫어해요? |

### 핵심 어휘

**🦁 동물 (3개)**
| 한국어 | English |
|--------|---------|
| 사자 | lion 🦁 |
| 기린 | giraffe 🦒 |
| 코끼리 | elephant 🐘 |

**🍓 과일 (3개)**
| 한국어 | English |
|--------|---------|
| 사과 | apple 🍎 |
| 딸기 | strawberry 🍓 |
| 수박 | watermelon 🍉 |

**🍖 한국 음식 (3개)**
| 한국어 | English |
|--------|---------|
| 불고기 | bulgogi (Korean BBQ) 🥩 |
| 잡채 | japchae (glass noodles) 🍜 |
| 떡 | tteok (rice cake) 🍡 |

### 대화 예시
```
리암: 뭘 좋아해요?
애라: 저는 사과를 좋아해요.

올리비아: 뭘 싫어해요?
마야: 저는 떡을 싫어해요.

카요: 불고기를 좋아해요?
루카스: 네, 좋아해요! / 아니요, 싫어해요.
```

### 게임 아이디어
- **좋아요/싫어요 그래프**: 반 친구들이 좋아하는 과일/음식 조사해서 막대 그래프 만들기
- **릴레이 인터뷰**: "뭘 좋아해요?" 돌아가며 물어보고 적기
- **한국 음식 소개**: 불고기/잡채/떡 사진 보여주며 문화 이야기

### 문화 팁
> 🇰🇷 **불고기**는 달콤하게 양념한 소고기, **잡채**는 유리면 + 야채, **떡**은 쌀로 만든 쫄깃한 음식이에요. 한국에 가면 꼭 먹어봐요!

## 10과 — 종합 연습 (복습) 🏆

**학습 목표**: 1~9과에서 배운 모든 표현을 복습한다

### 복습 포인트

| 과 | 핵심 질문 | 핵심 표현 |
|----|---------|---------|
| 1과 | 이름이 뭐예요? | 저는 ___이에요/예요. |
| 2과 | 이게/저게 뭐예요? | ___이에요/예요. |
| 3과 | 이게 ___이에요? | 네 / 아니요, ___이에요. |
| 4과 | 누구예요? | 우리 ___이에요/예요. |
| 5과 | ___이/가 어디예요? | 여기/저기예요. |
| 6과 | 어디에 가요? | ___에 가요. |
| 7과 | ___에 뭐가 있어요? | ___이/가 있어요. |
| 8과 | 뭘 해요? | ___을/를 ___해요. |
| 9과 | 뭘 좋아해요? | ___을/를 좋아해요/싫어해요. |

### 종합 활동

**빙고 게임 단어 풀 (25개)**
피아노 / 놀이터 / 색종이 / 빵 / 시계 / 사과 / 칠판 / 침대 / 크레파스 / 수박 / 아이스크림 / 딸기 / 텔레비전 / 강아지 / 토끼 / 냉장고 / 고양이 / 의자 / 공책 / 우유 / 책 / 컴퓨터 / 연필 / 지우개 / 식탁

**인터뷰 활동 — 전체 복습**
1. 이름이 뭐예요?
2. 이게 ___예요/이에요?
3. 누구예요?
4. 이게/저게 뭐예요?
5. 어디에 가요?
6. 뭘 해요?

## 전체 문법 체계 정리

| 문법 요소 | 처음 등장 | 형태 |
|----------|---------|------|
| N이에요/예요 | 1과 | 받침O → 이에요 / 받침X → 예요 |
| 이게/저게 | 2과 | 가까운 것 이게, 먼 것 저게 |
| 네/아니요 | 3과 | Yes/No |
| 누구예요? | 4과 | Who is it? |
| 어디예요? | 5과 | Where is it? |
| 여기/저기 | 5과 | here/there |
| N은/는 (topic) | 6과 | 받침O → 은 / 받침X → 는 |
| N에 가요/와요 | 6과 | go to / come to |
| N이/가 있어요 | 7과 | There is N (받침O → 이 / 받침X → 가) |
| 뭘 해요? | 8과 | What are you doing? |
| N을/를 (object) | 8과 | 받침O → 을 / 받침X → 를 |
| 좋아해요/싫어해요 | 9과 | like/don't like |

## 앱 구현 우선순위

### 📌 각 과에 공통으로 들어갈 탭 구조 (제안)
```
📚 새단어 Vocab  |  📌 문법 Grammar  |  ✏️ 연습 Practice
```

### 📌 Level 1 특별 고려사항 (미취학 아동)
1. **모든 한국어에 영어 병기** 필수
2. **이모지 적극 활용** — 그림이 설명보다 중요
3. **동사 게임 최소화** — 타이핑 없음, 클릭/드래그만
4. **짧은 대화** — 2~3줄 이상 넘기지 않기
5. **반복 패턴** — 매 과마다 같은 구조로 자신감 형성
6. **발음 힌트** — 로마자 표기 또는 영어 유사음 병기 고려
