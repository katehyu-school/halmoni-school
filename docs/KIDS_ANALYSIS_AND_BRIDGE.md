### 각 단원 설계 노트

**Unit 01 — -고 있어요/-고 계세요**  
Kids에서 완전히 빠진 진행형. 카드 2개: ①-고 있어요(일반) ②-고 계세요(존댓말 — u06 예고). "지금 뭐 해요? → 밥 먹고 있어요 / 할머니는 주무시고 계세요."

**Unit 02 — -(으)러 가다 + (으)로**  
"이동 표현" 단일 맥락으로 묶음. -(으)러 가다(왜 가요?)와 (으)로(어떻게 가요?)를 한 장면에서 대조: "도서관에 책 빌리러 가요. 자전거로 가요!"

**Unit 04 — 접속사 + ~의/만 경량**  
접속사가 핵심. ~의("엄마의 가방", 구어 생략 가능 note)와 만("하나만, 오늘만")은 각 예문 2~3개로 경량 처리.

**Unit 06 — 존댓말 -시- 단독 🔴**  
Kids 트랙 최초 높임법 본격 정리. 이미 알고 있는 `-세요`(주세요/앉으세요)에서 패턴 재인식 시작:
- -(으)시- 어간 변화: 가시다 / 드시다 / 계시다 (3개 핵심만)
- 과거 -(으)셨어요 (L3 u09 preview → 여기서 정착)
- u07 드리다와 분리해서 이 단원은 -시-에만 집중

**Unit 07 — 드리다 + 날짜**  
u05 -(으)ㄹ게요 → u06 -시- → u07 드리다로 자연 흐름. "선생님 생신 선물을 드릴게요!" 문장 하나에 세 단원이 녹아 있음. 날짜는 "몇 월 며칠이세요?" 맥락으로 자연 결합.

**Unit 09·10 — 관형절 1·2 (2단원 배분)**  
- u09: 형용사 관형(예쁜/큰/맛있는) → 이미 들어본 패턴부터 시작. 동사 현재형(-는) 추가.
- u10: 동사 과거형(-은/ㄴ) → 동사 미래형(-을/ㄹ). "어제 먹은 케이크 / 내일 마실 물"
- 2단원에 걸쳐 나선형 확장. 각 단원 step1/step2 게임으로 체득.

**Unit 12 — HQ 브릿지**  
마지막 화면: "이제 Hangeul Quest를 시작할 준비가 됐어요! 🎉"  
`[Hangeul Quest 시작하기]` 버튼 → `nhs.html` 직접 링크.
