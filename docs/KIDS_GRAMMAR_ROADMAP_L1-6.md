### → unit10(ep10)이 떠안게 된 "잔챙이" 목록

unit10 원래 계획(도/만/부터~까지) 중 **부터~까지는 unit09에서 이미 처리됨**. 대신 07·09에서 빠진 항목들이 전부 unit10으로 떠넘겨진 상태:

| 항목 | 원래 위치 | 비고 |
|------|----------|------|
| ~의 소유격 | unit07 예정 → 빠짐 | 구어에서 자주 생략 — 가벼운 처리로 충분 |
| N한테/한테서 | unit09 예정 → 빠짐 | 핵심 문법, ep10 스크립트의 중심 |
| 왜/어떻게 | unit06·08 예정 → 둘 다 빠짐 | 의문사, 비교적 가벼움 |
| 도(also) 독립 패턴 | unit10 원래 계획 | ep10 스크립트에 등장 ("팔찌도") |
| 만(only) | unit10 원래 계획 | ep10 스크립트에 없음 |
| _로 (에게, 문어체) | unit09 예정 → 빠짐 | 우선순위 낮음 |

## 🎯 unit10 (ep10) 최종 문법 확정 (2026-06-15)

> 선생님 스크립트 분석 결과: 위 잔챙이 중 **의/한테·한테서/왜·어떻게/도** 4개가 스크립트에 자연스럽게 등장.
> **만(only), _로**는 스크립트에 없고 억지로 넣으면 과부하 → **Level 4로 이연**.
> 스크립트에서 발견된 **"~도 돼요?" (허락 구하기)**는 원래 계획에 없었지만 unit06 "-(으)면 안 돼요"의 짝꿍 표현이자 이 에피소드의 진짜 주제라 core로 추가.
> "주실 거예요"(존댓말+미래)는 Kids 트랙 첫 존댓말 노출 — 본격 학습은 Level 4로, 여기서는 "눈치채기" 프리뷰만.

### 확정 카드 구성 (4 cards, tier 적용)

| tier | 카드 | 내용 | 출처 |
|------|------|------|------|
| core | g10_doedwaeyo | ~도 돼요? / -(으)면 안 돼요 (복습) — 허락 구하기·주기 | 신규 발견 (스크립트 핵심) |
| core | g10_hante_hanteseo | N한테 / N한테서 — ~에게/~로부터 | unit09 잔챙이 |
| preview | g10_wae_eotteoke | 왜? / 어떻게? + ~도 (나열: ~도 하고 ~도) | unit06·08·10 잔챙이 |
| tip (다른 카드 안에 포함) | ~의 소유격 | "할머니(의) 집" — 구어에서 생략 가능 note | unit07 잔챙이 |

### Level 4로 이연되는 항목
- **만 (only)** — 적절한 단원에서 자연 노출 시 배치 (예: 비교/선택 주제)
- **_로 (에게, 문어체)** — 우선순위 낮음, 필요 시 배치
- **주실 거예요 (존댓말 -시- + 거예요)** — Level 4 존댓말 도입 단원에서 본격 학습. ep10에서는 "Did you notice?" 프리뷰 노트만

> ⚠️ 03 "왜 울어요?"는 unit10에서 왜/어떻게를 preview로 다뤘으므로 — L4에서는 심화(접속사 연결)로 자연스러운 나선형 복습. 04 "-(으)면 안 돼요"도 unit10과 중복되나 복습으로 명시되어 있어 OK.
> 존댓말(-시-) 본격 도입 위치는 L4 설계 시 확정 필요 (현재 미배치).
