## ✅ Level 1 — 이미 노출된 문법 (nhs.html ep01~12)

| ep | 장면 | 핵심 문법 |
|----|------|----------|
| ep01 | 공원 | 은/는 (topic), N이에요/예요 |
| ep02 | 미래네 집 | 이/가 (subject), N이/가 아니에요 |
| ep03 | 저녁 식사 | 을/를 (object), -아요/어요 (현재), 이거/그거/저거 |
| ep04 | 미래의 방 | 에 (time/place), 에서 (activity), 몇 시, 네/아니요 |
| ep05 | 버스 정류장 | -(으)러 가다, -(으)ㄹ까?, 안+동사, -아/어서 |
| ep06 | 포장마차 | 하고/(이)랑/와·과, 뭐/무엇, N인분 |
| ep07 | 학교 교실 | -고 싶다, -기도 하다, 못+동사 (preview) |
| ep08 | 미래네 부엌 | 어디 있어요?, 위치+에 있어요, -(아/어)야 해요 |
| ep09 | 재래시장 | 고유어 숫자, 고유어 vs 한자어 |
| ep10 | 한강공원 | -았/었어요 (과거), -고 있어요/-고 계세요 (진행) |
| ep11 | 리암네 집 | -(으)셨어요 (높임 과거), -아/어야지 |
| ep12 | 경복궁 | -(으)ㄹ 거예요 (미래), -(으)ㄹ 수 있어요, 의 |

**L1 총 문법 목록**: 은/는, 이/가, 을/를, 에, 에서, N이에요/예요, N아니에요, 아요/어요, 있어요/없어요, 았/었어요, (으)ㄹ 거예요, (으)ㄹ 수 있어요, 고 있어요/계세요, 못+동사, 안+동사, (으)로, 고 싶다, 기도 하다, 아/어서(이유/순서), (으)러 가다, (으)ㄹ까?, 아/어야 해요, 아/어야지, (으)셨어요, 고유어/한자어 숫자, 단위명사, 의, 위치표현 (위/아래/앞/뒤/옆/안)

## 🗺️ Level 2 — 플랜 (12 에피소드 · TOPIK 2 · A2)

> 핵심 전략: L1의 단문 → 복문 연결. "왜?", "언제?", "어떻게?" 확장  
> ✅ ep01~03은 콘텐츠(슬라이드/TTS) 완성, ep04~12 플랜

| ep | 장면 | 문법 1 | 문법 2 | 문법 3 |
|----|------|--------|--------|--------|
| ep01 ✅ | 🛬 인천공항 도착 | -아야 돼 (have to) | -(으)면 (if ~) | 못+동사 심화 |
| ep02 ✅ | 🚌 공항버스 | -는 N (현재 관형절) | -(으)면 돼 (it's okay if) | -(으)ㄹ게요 (I will) |
| ep03 ✅ | 🤒 개울·감기 | -니까 (because, reason) | -(으)ㄹ 것 같아요 (seems/I think) | -기 (명사화 입문) |
| ep04 | 🏥 병원 진료 | -아/어 보다 (try doing) | -(으)ㄴ 것 같아요 (seems like, past) | -지만 (but, 역접) |
| ep05 | 🛒 재래시장 쇼핑 | -(으)ㄴ N (과거 관형절) | -(으)ㄹ N (미래 관형절) | -는데 (background context) |
| ep06 | 🍳 할머니 요리 | -(으)면서 (while ~ing) | -기 전에 (before) | -(으)ㄴ 후에 (after) |
| ep07 | 📱 친구들과 연락 | -(으)려고 하다 (intend to) | -(으)ㄹ래요 (want to? / shall we?) | -아/어 달라고 하다 |
| ep08 | 🏙️ 서울 나들이 | -아/어지다 (become ~) | -게 되다 (come to ~) | -는 중이에요 (in the middle of) |
| ep09 | 🎎 한국 명절 | -거든요 (you see / explanation) | -잖아요 (you know) | -(으)ㄹ 때 (when ~, time) |
| ep10 | 🌧️ 비 오는 날 | -아/어도 (even if) | -더라도 (even if, stronger) | -아/어야겠다 (I should/must) |
| ep11 | 🎤 노래방 | -다고 하다 (간접화법 입문) | -(이)라고 하다 | -고 싶다고 하다 |
| ep12 | ✈️ 귀국 준비 | -아/어 버리다 (do completely) | -아/어 놓다 (do and leave state) | 복습 + L2 마감 테스트 |

| ep | 테마 | 문법 1 | 문법 2 | 문법 3 |
|----|------|--------|--------|--------|
| ep01 | 소문 | -다고 하다 (direct quote) | -(이)라고 하다 | -냐고 하다 (indirect question) |
| ep02 | 부탁과 거절 | -(으)라고 하다 (indirect command) | -지 말라고 하다 | -아/어 달라고 하다 |
| ep03 | 뉴스 읽기 | 피동 -이/히/리/기 | -아/어지다 (피동적 변화) | 능동 vs 피동 비교 |
| ep04 | 요리 만들기 | 사동 -이/히/리/기/우 | -게 하다 (causative) | -도록 하다 |

| ep06 | 양보와 역접 | -(으)ㄴ데도/는데도 (even though) | -아/어도 심화 | -더라도 (even if) |
| ep07 | 목적과 조건 | -기 위해서 (in order to) | -(으)려면 (if you want to) | -(으)ㄹ 수록 (the more ~) |
| ep08 | 비교와 기준 | -에 비해서 (compared to) | -에 따라서 (depending on) | -처럼/같이 (like, as) |
| ep09 | 가능성 판단 | -(으)ㄹ 만하다 (worth ~ing) | -(으)ㄹ 만큼 (as much as) | -(으)ㄹ 뻔했다 (almost did) |
| ep10 | 대조와 대안 | -는 반면에 (whereas) | -는 대신에 (instead of) | -(으)ㄴ/는 편이다 (tend to) |
| ep11 | 문어체 입문 | -ㄴ다/는다 (plain form) | -이다 (서술) | -(으)ㄴ/는 것이다 (formal it is) |

## 📌 설계 원칙

1. **나선형 복습**: 중요 문법은 3~4 에피소드마다 다른 맥락으로 재등장
2. **Scene-First**: 문법 먼저가 아니라 장면에서 자연스럽게 노출
3. **3개 규칙**: 에피소드당 core 3개 — 하나는 신규, 하나는 확장, 하나는 복습
4. **TOPIK 연계**: 각 레벨 마감 테스트는 해당 급 TOPIK 실전 수준으로 설계
5. **읽기 우선**: CLAUDE.md TTS 습관 문제 반영 — 모든 레벨에 reading 문항 포함
