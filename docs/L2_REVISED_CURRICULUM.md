### ✅ L2 ep01~05 커버된 문법 체크리스트
- 관형절 3형 완성: -(으)ㄴ N (ep01), -는 N (ep01), -(으)ㄹ N (ep05) ✅
- -(으)면 조건 ✅ / 못+동사 심화 ✅ / 안+동사 ✅ / 안 vs 못 비교 ✅
- -(으)ㄹ게요 ✅ / -아/어 보다 ✅
- -(으)ㄴ 것 같아요 ✅ / -(으)ㄹ 것 같아 ✅
- -(으)니까 ✅ / -아/어서(이유) ✅ ← L1 순서 의미와 구별
- -아/어도 ✅ / -지만 ✅ / -는데 ✅
- -에게/-한테 ✅ (보너스)

## 🗺️ Section 3 — L2 ep06~12 재설계 (확정안)

> 설계 원칙:  
> ① 에피소드당 core 3개 + 보조 1~2개  
> ② 테마와 문법이 자연스럽게 연결  
> ③ 각 에피소드는 이전 ep를 나선형으로 복습  
> ④ L3 간접화법/피동 준비를 마지막 2에피소드에서 부드럽게 예열  
> **⑤ 경중 조절 원칙**: 에피소드 안에서 문법 항목의 무게를 균형 있게 배분  

### ⚖️ 문법 경중 조절 원칙

| 무게 | 기준 | 예시 |
|------|------|------|
| 🔴 Heavy | 새 체계 전체 (패러다임), 개념의 도약 | -(으)시- 전체, 간접화법, 명사화 |
| 🟡 Medium | 새 패턴이지만 형태 변화 단순 | -기 때문에, -(으)려고 하다, -게 되다 |
| 🟢 Light | 자연 짝꿍·연장·뉘앙스 추가 | 보다, -한테서, -네요, -(으)면 돼 |

**규칙**: 에피소드에 🔴가 하나 있으면 나머지는 🟡+🟢 조합. 🔴 둘은 절대 금지. 🟡 셋도 가능하면 피함.

| ep | 무게 조합 | 비고 |
|----|----------|------|
| ep06 | 🔴 -(으)시- + 🟡 -기 때문에 + 🟢 보다 | -(으)시- 중심 에피소드 |
| ep07 | 🟡 -(으)려고 하다 + 🟡 -(으)ㄹ래요? + 🟡 -아/어 주다 | 균등 Medium |
| ep08 | 🟡 -기 전에/후에 + 🟢 -(으)면서 + 🟢 -(으)ㄹ 때 | 짝꿍 묶음 — 실질 1.5개 |
| ep09 | 🟡 -아/어지다 + 🟡 -게 되다 + 🟡 -는 것 | 균등 Medium (테마 묶음) |
| ep10 | 🟢 -거든요 + 🟢 -잖아요 심화 + 🟡 -아/어야겠다 | 구어 뉘앙스 에피소드 — 가볍게 |
| ep11 | 🔴 -다고 하다 + 🟢 -(이)라고 하다 + 🟢 -아/어 버리다/놓다 | 간접화법 중심, 나머지 가볍게 |
| ep12 | 복습 | 신규 문법 없음 |

### ep06 — 🏠 할머니 댁에 왔어요
**장면**: 할머니·할아버지 댁 방문. 어른과 대화하는 상황 → 높임법 가장 자연스러운 컨텍스트

| tier | 문법 | 예시 |
|------|------|------|
| **core** | `-(으)시-` 높임 선어말어미 체계 | 드세요 / 드셨어요 / 드실 거예요 — 패러다임 완성 |
| **core** | `보다` 비교 | "할머니 손이 저보다 따뜻해요" |
| **core** | `-기 때문에` / `N 때문에` | "날이 춥기 때문에 따뜻하게 입었어요" |
| 보조 | `-아/어 드리다` | "제가 가방 들어 드릴게요" (주다 → 드리다, -(으)시- 연결) |
| 보조 | `-(으)면 돼` (복습 확인) | "이렇게 하면 돼요?" |

> 📌 ep11(L1)에서 -(으)셨어요를 먼저 노출했기 때문에 ep06에서는 "기억하죠? 이제 전체 체계를 배워요" 방식.  
> -(으)시- 체계 카드: 현재(세요)/과거(셨어요)/미래(실 거예요)/진행(고 계세요) 패러다임 한 눈에.

### ep07 — 🎵 같이 뭐 할까요?
**장면**: 친구들과 주말 계획 잡기, 약속 대화 → 의도·제안·부탁 표현 묶음

| tier | 문법 | 예시 |
|------|------|------|
| **core** | `-(으)려고 하다` 의도 | "이번 주말에 한강에 가려고 해요" |
| **core** | `-(으)ㄹ래요?` 제안·의향 | "같이 갈래요? / 저도 갈래요!" |
| **core** | `-아/어 주다 / 드리다` | "이 가방 들어 줄게요 / 선생님께 드렸어요" |
| 보조 | `-한테서` (from) | "리아한테서 문자 왔어요" (ep05 -한테 짝) |
| 보조 | `-지 못하다` 능력 부정 | "못 가 → 가지 못해요" (격식형, 못+V와 비교) |

> 📌 `-아/어 주다/드리다` 비교 카드: 주다(친구), 드리다(어른) — ep06 드리다와 연결.  
> `-(으)ㄹ래요?` vs `-(으)ㄹ까요?` 비교 note 포함 (L1 ep05 복습).

### ep08 — 🍳 요리 순서를 알아요
**장면**: 한국 음식 만들기, 순서와 동시 행동 → 시간 관계 표현 집중 에피소드

| tier | 문법 | 예시 |
|------|------|------|
| **core** | `-기 전에` / `-(으)ㄴ 후에` | "씻기 전에 손을 씻어요 / 먹은 후에 설거지해요" |
| **core** | `-(으)면서` 동시 행동 | "음악을 들으면서 요리해요" |
| **core** | `-(으)ㄹ 때` 시간적 배경 | "배고플 때 이거 먹어요" |
| 보조 | `-는 동안` 기간 | "기다리는 동안 뭐 할까요?" |
| 보조 | `-기 시작하다` | "한국어를 배우기 시작했어요" |

> 📌 시간 표현 비교 카드 권장: `-기 전에` / `-(으)ㄴ 후에` / `-(으)ㄹ 때` / `-는 동안` 한 눈에 정리.

### ep09 — 🌱 달라졌어요!
**장면**: 한국 생활 후 변화된 것들, 감정과 상태 묘사 → 변화 표현 집중

| tier | 문법 | 예시 |
|------|------|------|
| **core** | `-아/어지다` 상태 변화 | "한국어가 쉬워졌어요 / 날이 추워졌어요" |
| **core** | `-게 되다` 피동적 변화·결과 | "한국을 좋아하게 됐어요 / 알게 됐어요" |
| **core** | `-는 것` 명사화 | "한국어를 배우는 것이 재미있어요" |
| 보조 | `-는 것 같다` (현재 추측) | "피곤한 것 같아요" ← ep03 -(으)ㄴ것 같아 세트 완성 |
| 보조 | `-(으)면 돼` 확인 | "이렇게 하면 돼요?" (ep02 미포함분 마무리) |

> 📌 `-아/어지다` vs `-게 되다` 비교 카드 필수:  
> - 지다 = 상태가 스스로 변함 (날이 더워졌어요)  
> - 게 되다 = 상황에 의해 결과가 생김 (한국에 살게 됐어요)  
> 📌 명사화 완성: -(으)ㄴ 것 (ep03) + -는 것 (ep09) + -(으)ㄹ 것 (ep04 -(으)ㄹ 것 같아) = 3형 완성

### ep10 — 💬 솔직히 말해봐요
**장면**: 친구들과 솔직한 대화, 감탄·발견·의지 표현 → 구어 표현 집중 에피소드

| tier | 문법 | 예시 |
|------|------|------|
| **core** | `-거든요` 배경 설명 | "저는 못 가거든요. 일이 있어서요." |
| **core** | `-잖아요` 심화 | "알잖아요! / 한국이 좋잖아요~" (ep05 preview 심화) |
| **core** | `-아/어야겠다` 강한 의지 | "더 열심히 해야겠어요" (L1 -아/어야 해요 업그레이드) |
| 보조 | `-네요` 감탄·새발견 | "와, 맛있네요! / 한국어가 늘었네요!" |
| 보조 | `-았/었으면 좋겠다` | "눈이 왔으면 좋겠어요" |

> 📌 `-아/어야 해요` (L1 ep08) vs `-아/어야겠다` (L2 ep10) 비교 note:  
> - -야 해요 = 외부 의무 "해야만 해" (규칙, 상황)  
> - -야겠다 = 자기 결심 "나는 해야지" (내부 의지)  
> 📌 `-네요` = 새로운 사실을 깨달을 때. "정말 맛있네요!" Sejong 1A U10에서 왔지만 L1 미포함 → 여기서 처리.

### ep11 — 📢 들은 이야기예요
**장면**: 누군가에게 들은 이야기 전달, 소문 이야기 → 간접화법 입문 에피소드 (L3 예열)

| tier | 문법 | 예시 |
|------|------|------|
| **core** | `-다고 하다` 평서문 간접화법 | "미래가 한국어를 잘한다고 했어요" |
| **core** | `-(이)라고 하다` 명사 간접화법 | "이름이 리암이라고 했어요" |
| **core** | `-아/어 버리다` / `-아/어 놓다` | "다 먹어 버렸어요!" / "미리 사 놨어요" |
| 보조 | `-고 싶다고 하다` | "한국에 가고 싶다고 했어요" |
| 보조 | `-더라도` 양보 | "비가 오더라도 갈 거예요" [preview for L3] |

> 📌 간접화법은 L3 ep01-03에서 본격 완성. 여기서는 "평서문"과 "명사"만.  
> 📌 `-아/어 버리다` vs `-아/어 놓다` 비교:  
> - 버리다 = 완전히 끝냄 (아쉬움/시원함: "다 먹어 버렸어!")  
> - 놓다 = 미리 해두기 (준비: "예약해 놨어요")

## 📊 Section 4 — L2 ep01~12 완성 후 문법 목록

> 이 목록이 L3 시작 전 학생의 보유 문법이 됨.

### 조사
은/는, 이/가, 을/를, 에(시간/장소), 에서(행동), (으)로, 하고/(이)랑/와·과, 에게/한테, 한테서, 의, 도, 보다

### 동사·형용사 활용
-아요/어요, -았/었어요, -(으)ㄹ 거예요, -고 있어요/계세요, -(으)셨어요, -(으)시- 체계

### 관형절 (3형 완성)
-(으)ㄴ+N, -는+N, -(으)ㄹ+N

### 추측·판단
-(으)ㄴ 것 같아요, -는 것 같아요, -(으)ㄹ 것 같아요

### 부정
안+V, 못+V, -지 않다, -지 못하다, N이/가 아니에요

### 연결어미
-아/어서(순서·이유), -(으)니까, -기 때문에, -는데, -(으)면, -아/어도, -지만, -(으)면서, -(으)ㄹ 때, -기 전에, -(으)ㄴ 후에, -는 동안, -더라도[preview]

### 보조동사
-고 싶다, -아/어 보다, -아/어 주다/드리다, -아/어지다, -게 되다, -아/어야 해요, -아/어야겠다, -기 시작하다, -아/어 버리다, -아/어 놓다, -(으)ㄹ 수 있어요/없어요

### 의도·계획
-(으)ㄹ게요, -(으)려고 하다, -(으)ㄹ래요?

### 명사화
-는 것, -(으)ㄴ 것, -기 때문에(명사형 인과)

### 간접화법 (입문)
-다고 하다, -(이)라고 하다, -고 싶다고 하다

### 구어 표현
-거든요, -잖아요, -네요, -(으)면 돼, -았/었으면 좋겠다
