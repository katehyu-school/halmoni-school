## 🎬 Hangeul Quest 에피소드 기획 (Original 장면 아이디어)

### ✅ 이미 완성된 에피소드
- **ep01** 🏞️ 공원·첫 만남 — 인사/자기소개 (unit 00-01 영역)
- **ep02** 🍚 저녁 식사 — 가족, 음식 (unit 04 영역 일부)
- **ep03** 🎢 놀이공원 매표소 — 숫자/가격 (unit 02, 06 영역 일부)
- **ep04** 🏪 포장마차 — 주문/단위/가격 (unit 06 영역, 자료 준비됨)

### 🚀 다음 에피소드 아이디어 (저작권 안전 + 한국적)

#### 🔢 unit02 영역 (연락처, 한자어 숫자) — 미커버
**ep 후보**: 📱 새 친구의 카톡 ID 받기
- 장면: 학교/카페에서 친구와 카톡 ID/전화번호 교환
- 캐릭터: 정민 + 새 친구 (미나 활용 가능)
- 핵심 표현: "카톡 아이디가 뭐야?", "○○-○○○○-○○○○", "찾아볼게"
- Real Life: kakao 타입 활용 → 자연스러운 카톡 대화

#### 📍 unit03 영역 (사물 위치) — 미커버
**ep 후보**: 🎒 잃어버린 가방 찾기
- 장면: 학교 도서관/카페에서 가방을 어디 뒀는지 못 찾는 상황
- 핵심 표현: "내 가방 어디 있어?", "책상 옆에", "의자 위에", "테이블 아래"
- 자연스럽게 이/그/저, 위치 명사 등장

#### 📚 unit04 영역 (일상 활동) — 미커버
**ep 후보**: 🏠 학교 다녀온 정민이의 하루
- 장면: 정민이가 집에 와서 엄마한테 오늘 하루 보고
- 핵심 표현: "한국어 공부해요", "친구를 만나요", "숙제를 해요"
- -아요/어요 자연 등장

#### 🛒 unit05 영역 (물건 사기) — 미커버
**ep 후보**: 🏪 편의점에서 야식 사기
- 장면: 정민/미래가 편의점 가서 라면+삼각김밥+우유 사기
- 핵심 표현: "○○하고 ○○ 주세요", "얼마예요?", "봉투 주세요"
- 한국적 편의점 문화 (포인트 적립, 봉투값) → tip 카드

#### ⏰ unit07 영역 (시간) — 미커버
**ep 후보**: 📞 영화 약속 정하기
- 장면: 미나와 정민이 영화 보러 가기로 약속 (전화 통화)
- 핵심 표현: "몇 시에 만날까?", "일곱 시 어때?", "그래 그때 봐"
- Real Life: listening 타입 (전화 메시지)

#### 🌤️ unit08 영역 (날씨) — 미커버
**ep 후보**: 🌸 봄 소풍 / 🌨️ 첫눈 오는 날
- 장면: 날씨에 따라 옷차림/계획 결정
- 핵심 표현: "오늘 날씨 어때?", "추워요/더워요", "비 와요"
- 한국 4계절 문화 → tip 카드 (장마, 황사 등)
- ㅂ 불규칙 동사 자연 노출

#### 🚴 unit09 영역 (주말 활동, 과거형) — 미커버
**ep 후보**: 🏞️ 한강 자전거 / 🎤 노래방
- 장면: 정민/미나가 어제 한강에서 자전거 탔던 이야기 미래한테 들려줌
- 핵심 표현: "어제 뭐 했어?", "한강에서 자전거 탔어", "정말 재미있었어"
- 과거형 -았/었어요 집중

#### 🎢 unit10 영역 (제안/약속) — 미커버
**ep 후보**: 🎉 친구 초대 — 우리 집 놀러 와!
- 장면: 미래가 친구들 자기 집 생일파티에 초대하는 상황
- 핵심 표현: "우리 집 올래?", "같이 놀까?", "내일 어때?"
- -(으)ㄹ까요?, 같이 자연 노출

## 📅 추천 에피소드 제작 순서

### Phase 1 (현재 완료/진행 중)
- ✅ ep01 (인사) — 완성
- ✅ ep02 (가족/식사) — 완성
- ✅ ep03 (가격/주문) — 완성
- 🔜 ep04 (포장마차) — 자료 준비됨, 작업 대기

### Phase 2 (다음 우선 - 일상생활)
- ep05: 🎒 학교에서 (잃어버린 가방 — 위치)
- ep06: 🏠 학교 다녀와서 (일상 활동)
- ep07: 🏪 편의점 야식 (쇼핑)

### Phase 3 (시간/약속)
- ep08: 📞 영화 약속 (시간)
- ep09: 🎉 생일파티 초대 (약속 잡기)

### Phase 4 (날씨/계절)
- ep10: 🌸 봄 소풍 (날씨)
- ep11: 🏞️ 한강 자전거 (주말 활동, 과거형)

### Phase 5 (확장)
- ep12+: 한국 문화 심화 (한복, 명절, K-pop 콘서트 등)
- Sejong에 없는 한국 현대 문화 활용

## 🎨 장면 다양성 체크리스트

각 ep는 다음 중 하나 이상의 한국적 장면을 포함하면 좋음:
- 🏞️ 공원 / 한강
- 🍱 분식집 / 포장마차 / 한식당
- 🏪 편의점 / 시장 / 백화점
- 🚇 지하철 / 버스
- 🎤 노래방 / PC방
- 🏫 학교 / 학원
- 🏠 한국 가정집 (현관에서 신발 벗기 등)
- ⛰️ 산 / 사찰
- 🎢 놀이동산 / 한강공원
- 💆 찜질방

→ 단순 학습이 아닌 **한국 생활/문화 체험**으로의 확장
